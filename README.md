# Films Locations .

Films Locations is a dynamic platform designed to track, explore, and discover the real-world filming locations of your favorite movies and TV shows. Whether you are  planning a movie-themed road trip or a developer looking for a comprehensive movie location database, Films Locations provides a seamless interactive experience.

---

##  Features
-**Index.html** is the home page.

-**Gallery.html** is a page that shows various movie locations.

-**Price.html** show the prices of various locations per hour.

-**Bookdate.html** allows users to book date of filming.

-**.js** are used to make the application responsive.
